# OpenHands Chat App 💬

AI-powered chat application powered by OpenHands.

## ✨ Features

- 🤖 AI chat assistant
- 💬 Real-time messaging
- 📁 File sharing
- 🔔 Notifications
- 🌐 Multi-language support

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/antono4/openhands-chat-app.git

# Open in browser
cd openhands-chat-app
open index.html
```

## 📁 Project Structure

```
openhands-chat-app/
├── index.html      # Main page
├── css/            # Stylesheets
├── js/             # JavaScript files
└── assets/         # Static assets
```
